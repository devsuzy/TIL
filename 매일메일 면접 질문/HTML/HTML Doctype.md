# HTML Doctype

> 매일메일 FE 2025년 3월 4주차 금요일 질문

### Doctype
```html
<!doctype html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <title>Hello World!</title>
  </head>
  <body></body>
</html>
```
- 웹 브라우저에 해당 문서가 어떤 HTML 버전을 기반으로 작성되었는지 알려주는 역할을 하는 선언문
- 문서의 최상단에 위치
- 브라우저가 HTML 문서를 해석하고 렌더링하는 방식을 결정
- Doctype을 선언함으로써 브라우저가 문서를 렌더링 할 때, 쿼크 모드(quirks mode)로 바뀌지 않도록 함
