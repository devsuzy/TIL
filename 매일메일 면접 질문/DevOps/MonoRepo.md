# MonoRepo

> 매일메일 FE 2025년 11월 4주차 월요일 질문

### 모노레포란?
- mono(단일)와 repo(레포지토리)를 합친 용어
- 독립적인 여러 프로젝트 또는 패키지를 하나의 코드 저장소에 통합하여 관리하는 방식

### 모노레포 설정 도구
- Lerna
- Yarn Workspaces 또는 Npm Workspace

### 장점
- 프로젝트의 코드 및 설정 등을 일관되게 관리 가능
- 의존성 관리를 한 곳에서 하기 때문에 프로젝트 내에서 공통으로 사용되는 유틸리티나 컴포넌트를 한 번 업데이트하면 모든 관련 프로젝트에 즉시 적용됨
-  여러 프로젝트들이 동일한 버전 관리 시스템을 공유

### 단점
- CI/CD 파이프라인 관리의 복잡도가 증가
- 특정 프로젝트만 권한 설정 및 권한 관리가 어려움
